# Gestión de Direcciones - Proyecto Full Stack

Este es un proyecto práctico desarrollado para la gestión integral de direcciones de clientes en una aplicación web. El objetivo principal fue implementar un sistema de autenticación seguro y operaciones CRUD sobre una base de datos documental.

## 🚀 Características principales

* **Autenticación segura:** Implementación de login con JWT (access y refresh tokens) para proteger las rutas.
* **Gestión de direcciones:**
    * Creación, edición y eliminación de direcciones de envío y facturación.
    * Funcionalidad para marcar direcciones como "principales" o de "facturación".
* **Persistencia:** Integración completa con MongoDB para el almacenamiento de datos.
* **Frontend dinámico:** Interfaz desarrollada en React con componentes reutilizables y formularios validados.

## 🛠️ Tecnologías utilizadas

* **Frontend:** React, Vite, CSS Modules.
* **Backend:** Node.js, Express, Mongoose.
* **Base de datos:** MongoDB.
* **Autenticación:** JWT (jsonwebtoken).
